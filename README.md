## Mobile friendly groceries application

## Front-end:
- AngularJS
- Bootstrap
- Font-Awesome
- Basic app routing

## Back-end:
- Node.js
- Express.js REST api and routing
- MongoDB and mongoose

## Pre-Requisites:
- MongoDB
- NPM

## How to Install:
- npm install

## How to run:
- npm start
- Visit localhost:8080 in browser
